Server Requirements

a HTTP web-server of some description
Apache 2.x, Abyss Web Server, Uniform Server, Zend Community Server, etc. So far PyroCMS only seems to struggle with IIS and nginx.
MySQL 5.x or higher
There are plans for other database systems, but for now the newer MySQL version the better.
PHP 5.2.x or higher
PHP 5.2 and PHP 5.3 are both fully supported.
GD 2
This library will need to be installed on your server, and PHP will need to be compiled to include it. Details on how to do that can be found here.
cURL
libcurl 7.10.5 or greater will need to be installed on your server and PHP will need to be compiled to include it. Details on how to do that can be found here.

Bundled requirements

PyroCMS uses these third-party packages to run, but includes them in each release so you do not need to install them yourself.

  • CodeIgniter 2.0-dev
  • jQuery 1.4.2
  • Dwoo Template Engine 1.1.0
