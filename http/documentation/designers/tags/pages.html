pyro:pages

The pages tag displays page info.

Tags

{pyro:pages:url}


Gets a page's URL based on its ID.

Parameters

Parameter Default Description
id Required. Page ID the URL the tag should return.

Example

{pyro:pages:url id="4"}â€‹
http://www.example.com/about

{pyro:pages:children}


Tag pair that loops through the children of a parent page.

Parameters

Parameter Default Description
id Required. Page ID of the page you want the children for.
limit Required. Number of items to return.

Example

{pyro:pages:children id="1" limit="2"}
	<h2>{title}</h2>
	{body}
{/pyro:pages:children}
<h2>Child One</h2>
Body Content One
	
<h2>Child Two</h2>
Body Content Two

{pyro:pages:display}


A tag that allows any one page to be displayed inside other content.

Parameters

Parameter Default Description
id ID of the page you want to display.
slug Slug of the page you want to display.

Example

{pyro:pages:display slug="home"}
	<h2>{title}</h2>
	{body}
{/pyro:pages:display}
<h2>Page Title</h2>
<p>Page Body</p>

{pyro:pages:is}


A tag that says if a page is direct child of another page or is descendent of another page.

Parameters

Parameter Default Description
child ID or Slug of the page you want check relation.
children Child separated by comma.
parent ID or Slug of the page to check if is parent of child.
descendent ID or Slug of the page to check if is descendent of child.

Example

{if '{pyro:pages:is child="ingredients" parent="cookbook"}' == true}
<p>
    Click here to see table of contents cookbook
    etc...
</p>
{/if}

Display:

Click here to see table of contents cookbook
etc...
{if '{pyro:pages:is child="terms-and-conditions" parent="information"}' == true}
<body class="terms information">
{else}
<body class="">
{/if}

Display:

<body class="terms information">
