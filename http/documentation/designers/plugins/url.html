Url Plugin

The url plugin gives you access to url data and logic.

The user plugin's slug is url, so it can be used like so:

{pyro:url:function}

Functions

{pyro:url:current}

Displays the full current URL.


{pyro:url:site}

Displays the full site URL.

Parameters

Parameter Description
uri Optional. The URI segments passed to the site_url function.

Example:

{pyro:helper:site uri="contact"}
http://www.example.com/contact

{pyro:url:base}

Displays the full site base URL.


{pyro:url:segments}

Displays a specific URL segment.

Parameter Description
segment Required. Number of the segment of the URL you want to use.
default A default value to use if nothing is in the URL segment specified.

Example:

{pyro:url:segments segment="1" default="home"}

{pyro:url:anchor}

Generates an anchor tag based on the supplied attributes. This is essentially a wrapper for the built in anchor() function found in the url helper.

Parameter Description
segments Required. The segments passed to the anchor function.
title Optional. The text that is displayed between the <a></a> tags. If omitted, the generated url will be displayed.
class Optional. A class to be assigned to the anchor tag.

Example:

{pyro:url:anchor segments="users/login" title="Login" class="login"}
